To-do List February 2024

This is a dynamic Todo List web application built using HTML, CSS, and JavaScript. It provides a user-friendly interface for adding, managing, and filtering tasks. Users can add new tasks by typing into the input field and pressing Enter or clicking the "+" button. Tasks can be marked as completed by checking the checkboxes, and they can be deleted individually by clicking the "x" button. Additionally, users can filter tasks by status (completed or pending) or delete all tasks at once. The application utilizes local storage to persist tasks even after refreshing the page, ensuring a seamless user experience.
